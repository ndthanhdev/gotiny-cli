# gotiny-cli

Using [gotiny.cc](https://gotiny.cc) the lightweight, fast, secure URL shortener from the command line.

## Installation

**Macos:**
```bash
brew install gotiny
```

**Linux:**  
Via binary, apt(wip)

**Windows:**
Via binary, choco(wip)

## Usage

**Simple:**  
```
gotiny google.com
https://gotiny.cc/xxxxxx
```

**Custom code:**  
```
gotiny google.com -c abcdef
https://gotiny.cc/abcdef
```

**Help:**  
```
gotiny -h
```
